# AI-Mental-Health-Analyzer

## Project Overview
The AI Mental Health Analyzer is a simple expert system developed using Prolog. It is designed to interact with users and identify their emotional state based on input provided. The system analyzes basic responses and classifies the user’s mood into categories such as happy, sad, or normal, and provides appropriate feedback.

---

## Objective
The main objective of this project is to demonstrate how Artificial Intelligence concepts, particularly rule-based systems, can be used to simulate human-like interaction and provide basic mental health support.

---

## Working Principle
The system works using predefined rules written in Prolog.  
- The user is asked questions or provides input  
- The system matches input with rules  
- Based on matching, it identifies the emotional state  
- A suitable response is displayed  

This approach is based on logic programming and pattern matching.

---

##  Features
- Simple and interactive command-line interface  
- Detects emotional state (happy, sad, normal)  
- Provides supportive and positive responses  
- Lightweight and easy to run  

---

## Technologies Used
- Prolog (SWI-Prolog recommended)  
- Logic Programming  
- Rule-Based AI System

##  How to Run
1. Install SWI-Prolog
2. Open terminal
3. Navigate to project folder
4. Run:
   swipl mental_health.pl
5. Enter your input when prompted
